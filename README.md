# AI Resume Analyzer (Web Version)

## Description
This is an AI-powered Resume Analyzer web application built using Python and Streamlit.  
It helps users analyze their resumes by checking for:

- Key sections (Education, Skills, Projects, Experience)  
- Keyword relevance  
- Resume score (0â€“100)  
- Suggestions to improve readability and completeness  

## Features
- PDF & DOCX resume upload  
- Section detection & keyword analysis  
- Resume scoring  
- Suggestions & improvements  

## Tech Stack
- Python  
- Streamlit (web interface)  
- pdfplumber (PDF extraction)  
- python-docx (Word extraction)
